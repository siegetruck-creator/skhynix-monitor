# SK하이닉스 ADR 프리미엄 모니터

국내 `000660.KS`와 미국 ADR `SKHYV`/`SKHY`를 비교해 환율 및 ADR 비율을 반영한 프리미엄을 보여주는 로컬 웹앱입니다.

## 실행

Node.js 18 이상이 필요합니다.

```bash
npm start
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 계산 기준

`ADR 이론가격 = 국내주식 가격 ÷ USD/KRW 환율 ÷ 10`

`프리미엄 = (실제 ADR 가격 − ADR 이론가격) ÷ ADR 이론가격 × 100`

ADR 티커는 2026년 7월 13일을 기준으로 `SKHYV`에서 `SKHY`를 우선 조회하며, 새 티커 조회 실패 시 이전 티커로 자동 폴백합니다.
